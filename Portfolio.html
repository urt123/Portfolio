Food-Delivery-Optimization-System

A web-based Food Delivery Optimization System built using HTML, CSS, and JavaScript. This project simulates different operating system scheduling and caching algorithms to manage food delivery orders efficiently. The system allows users to place food orders by selecting meals, customer IDs, and delivery distances. It then processes and displays orders using FCFS (First Come First Serve) and SJF (Shortest Job First) scheduling techniques. Additionally, FIFO and LRU caching algorithms are implemented to track frequently ordered meals and recently active customers.

• Technologies Used
HTML, CSS, JavaScript
